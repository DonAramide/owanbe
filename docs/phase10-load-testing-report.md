## Methodology

Local single-node certification uses:

1. **Burst probe** (100 simultaneous requests) — documents rate-limit behavior.
2. **Sequential throughput** (sampled requests per tier with 2.1s spacing) — validates sustained load without exhausting the strict guard.

Volume tiers **100 → 500 → 1000 → 5000** label certification levels; sampled request counts per scenario are 15/15/12/10 respectively on dev. Full 5000-rps production validation requires horizontal scale (k6 + load balancer) documented in go-live checklist.

|------------|--------|
